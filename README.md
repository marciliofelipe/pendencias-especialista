## 📋 Como Usar Este Dashboard

### Para Atualizar Status:
1. Abra a issue relacionada
2. Atualize a **label de status** (✅ Finalizada / 🗓️ Em Andamento / ⌚ Aguardando)
3. Comente com update e próximos passos
4. Mencione responsáveis se necessário

### Para Acompanhar Bloqueadores:
- Procure issues com label 😶‍🌫️ **Bloqueada**
- Cada issue tem seção de "Bloqueadores Atuais"
- Atualize conforme informações chegam

### Para Visualizar por Urgência:
- 🔴 **Crítica** - Requer ação hoje/semana
- 🟡 **Média** - Pode esperar até próxima semana
- 🟢 **Baixa** - Backlog ou investigação
